## Key Features of FileJuggler

- **Real-Time Folder Monitoring** – FileJuggler watches any folder you choose and reacts instantly to new or changed files. FileJuggler supports recursive subfolder scanning and multiple simultaneous watchers.  
- **Rule-Based Automation** – FileJuggler lets you create unlimited custom rules based on file name masks, file extensions, file size ranges, creation or modification dates, and regular expressions. Each rule in FileJuggler triggers specific actions like move, copy, rename, delete, or run a program.  
- **Batch Renaming Engine** – FileJuggler includes a powerful renaming module with variable tokens (date, counter, original name, parent folder name). FileJuggler can rename entire file collections in one click while preserving original timestamps.  
- **Scheduled Scans** – FileJuggler supports hourly, daily, or weekly deep scans for complex folder structures that require periodic cleaning. FileJuggler runs these scans as Windows scheduled tasks.  
- **Detailed Logging** – FileJuggler writes every action to a timestamped log file. FileJuggler can also send log summaries via email or save them to a network drive.  
- **Portable Mode** – FileJuggler can run from a USB stick without installation. FileJuggler saves all settings and rules inside its own folder, making FileJuggler perfect for technicians and on-the-go file management.  
- **Multi-Action Rules** – One single rule in FileJuggler can perform multiple actions: move a file, rename it, and then launch a script. FileJuggler executes actions in the order you define.  
- **Conditional Processing** – FileJuggler supports advanced conditions like "only process files older than 2 days" or "skip files currently open by another program". FileJuggler prevents conflicts with active documents.

## Quick Start with FileJuggler

1. **Get FileJuggler Windows** – Grab the portable `.exe` (no installation required) or the installer version of FileJuggler.
2. **Launch** – Double-click `FileJuggler_Windows.exe`. The main window of FileJuggler opens in under 1 second.
3. **Add your first watched folder** – Click "Add Folder", select a directory (for example `C:\Users\YourName\Downloads`), and close the dialog.
4. **Create your first rule** – Click "New Rule" → set condition "Extension equals .jpg" → set action "Move to C:\Sorted\Photos". Then click Save in FileJuggler.
5. **Test your rule** – Drop any `.jpg` file into the watched folder. Within 2 seconds, FileJuggler moves it automatically to the target folder.
6. **Adjust (optional)** – Change rule order, enable notifications, or set up email reports. Most people use FileJuggler defaults because they already work perfectly.
7. **Work normally** – FileJuggler activates automatically when you add files to watched folders. You don't need to think about FileJuggler — FileJuggler just works.
